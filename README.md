# Broker Financier

Projet 1 — Licence 3 MIASHS — Programmation système et réseau  
Université Toulouse Jean Jaurès — 2025/2026

## Description

Système client-serveur en C simulant un broker financier.  
Le serveur gère plusieurs clients simultanément via des threads et permet la consultation de produits financiers ainsi que l'achat et la vente d'actifs.

- Olti

## Compilation

```bash
make
```

## Exécution

Lancer le serveur :
```bash
./broker
```

Lancer un client (dans un autre terminal) :
```bash
./client
```

## Nettoyage

```bash
make clean
```
